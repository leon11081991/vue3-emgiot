## 程式碼風格規範

1. 參考 [Vue 官方程式碼風格規範 Style Guide](https://vuejs.org/style-guide)

2. 程式碼檢查及排版
   - [ESLint](https://eslint.org) - 檢查程式碼語法、統一編寫風格
